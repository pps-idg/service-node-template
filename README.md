# inventory-backend-node

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode from the build folder.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run dev`

Launches the project in dev with watch mode.<br />

### `npm run ts-node`

Launches the project using the tsc compiler.<br />

### `npm run build`

Builds the app for production to the `build` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.

## `Packages and dependencies`

- [Express](https://expressjs.com)
- [TypeScript](https://www.typescriptlang.org)

## `Dev dependencies`

- [TS-Node](https://www.npmjs.com/package/ts-node)
- [@Types](https://www.npmjs.com/package/@types/express)
- [Nodemon](https://www.npmjs.com/package/nodemon)

## `creation and Install procedure`

- npm init
- npm install express --save
- npm install -g typescript
- tsc --init
- npm i @types/express -D
- npm install -D typescript
- npm install -D ts-node
- npm install --save-dev nodemon
